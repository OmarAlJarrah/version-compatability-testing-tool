# version-compatibility-testing-tool

A tool for testing multiple programming language versions for a project.

## Requirements
+ Docker
+ Unix System


## Supported languages
### Java
+ JDKs
  + JDK 8
  + JDK 11
  + JDK 15
  + JDK 16
  + JDK 18

+ Dependency Managers
  + Gradle
  + Maven (to be supported)

### Other Languages
+ To be supported in future releases.
